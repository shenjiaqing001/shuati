[团建题目 1] 现在给两个相同长度的字符串s和t（长度都是1e5），都仅包含小写字母。你允许最多把s中的两处字符交换一次。问最多能构造多少匹配对，所谓的匹配对，就是s[i]==t[i]。

[团建题目 2] 我们假设所有的机场都是由三个大写字母表示。同时给出了一系列机票，机票上写着起飞和降落的机场的名字。现在给一个行程安排itinerary标记了依次会经过的机场，但这些机场的名字可能打印错误。问最少需要修改多少个itinerary上的字母，可以用所给机票拼接出与itinerary一致的行程路径。

[团建题目 3] 现在有N个人凌晨出发去迪士尼门口排队开门。已知每个人达到的时间段是[start,end]，即这个人可能在[start,end]这个双闭区间里的任意时刻到达并加入排队的人群。如果同时到达的若干人，他们加入队伍的先后顺序可以是任意的。先给出一个数字k。问每个人是否有可能恰好是队伍中的第k个人。

[团建题目 4] 给你一个数组nums。再给你一些列queries，每个query包含一个区间[i,j]，问nums[i:j]是否是一个等差数列。nums和queries的长度都是1e5.

[团建题目 5] 现在有最多1e9颗棋子，编号从0开始递增排列。我们要将其染成黑色或者白色。有以下几个方法可以调用：
void setBlack(int i) 把第i颗棋子染成黑色
void setWhite(int i) 把第i颗棋子染成白色
void setAllBlack() 把所有棋子染成黑色
void setAllWhile() 把所有棋子染成白色
int getValue(int i) 返回第i颗棋子的颜色（1代表黑，0代表白)

[团建题目 6] 给出一个数组nums。另外给出一系列query. 每次query时给定一个数字q，求该数组里有多少个不同的、长度为w的subarray，使得该subarray一定存在数字q。nums和queries的长度是1e5.
